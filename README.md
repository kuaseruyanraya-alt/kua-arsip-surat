# KUA Arsip Surat (MVP)

Repo ini adalah scaffolding minimal untuk aplikasi Arsip Surat KUA.
Isi:
- backend: Express API minimal dengan endpoint contoh dan template PDF (EJS)
- frontend: React (Vite-style) komponen minimal untuk form Surat TT Catin, util NIK/format
- docs: API & design notes

Ini hanya MVP starter. Silakan lengkapi sesuai kebutuhan.
